# Calculadora CI

Projeto desenvolvido para a disciplina de Integração Contínua.

## Tecnologias

- Java 21
- Spring Boot
- Maven
- JUnit 5

## Como executar

```bash
./mvnw spring-boot:run
```

A aplicação ficará disponível em:

```
http://localhost:8080
```

### Exemplos

```
/somar?a=5&b=3
```

Resultado:

```
8
```

```
/subtrair?a=10&b=4
```

Resultado:

```
6
```

```
/multiplicar?a=5&b=8
```

Resultado:

```
40
```

```
/dividir?a=20&b=5
```

Resultado:

```
4
```
